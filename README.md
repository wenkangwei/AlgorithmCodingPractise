# Study Goal
取决于自己的学习目标， 学习+实践的方法有很多种， 如果偏向于做AI应用，那么学习AI产品知识+AI工具调用即可。如果偏向于系统学习知识点，那么抽一部分主流的理论知识点学习，然后看业界落地的算法/功能怎么实现即可。

**而这个仓库的学习目标主要是系统性整理大模型/AI agent的一些知识点，并加以实践。以及知道主流的工具怎么用即可。注意: 并不是深入工具框架原理**

1. LLM/AI-Agent理论知识点
    - 学习和尝试实现业界主流算法

    - 学习主流工具/框架是怎么实现知识点， 自己写的话怎么去实现
    - 怎么上手使用对应的工具/框架
3. 应用实践
